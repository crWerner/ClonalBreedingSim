# ClonalBreedingSim

R scripts used in 

*Werner, C.R., Gaynor, R.C., Sargent, D.J. et al. **Genomic selection strategies for clonally propagated crops.** Theor Appl Genet 136, 74 (2023). https://doi.org/10.1007/s00122-023-04300-6*

Code was updated for compatibility with the current versions of AlphaSimR, replacing outdated functionality from the original Figshare release. 

Minor adjustments to simulation settings and population sizes were made to allow execution on standard local computers.

**How to run:** Select the simulation scenarios in '00_RUN_SIM.R' and run. All subsequent scripts and dependencies are called automatically.
